# Figma 深色主题

[English](./README.md) | [简体中文](./README.zh.md)

用于 Google Chrome 和 Microsoft Edge 浏览器的 Figma 网页版深色主题。

拓展程序会将所有 Figma.com 网站及编辑器的页面的更改为深色主题。

## 打包和贡献

首先需要安装 `zip` 包。

```bash
# Add executable permission
chmod +x build.sh

# Build
bash build.sh
```

打包成功后 `dist` 目录下查看文件。
